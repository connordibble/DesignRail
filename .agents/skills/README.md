# Installed Third-Party Skills

This directory is for installed external skills. DesignRail-owned instructions live in `agents/` and are the source of truth for project behavior.

## Installed Skills

- `vercel-react-best-practices`: optional support for React performance reviews and implementation guidance.
- `vercel-react-view-transitions`: optional support for polished motion and view transitions after core flows are stable.
- `web-design-guidelines`: optional UI review aid. Do not put remote guideline fetches in deterministic quality gates.

## Usage

Use these skills as supporting references only. DesignRail rules in `AGENTS.md` and `agents/*.SKILL.md` take precedence.
